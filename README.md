## ℹ️ About oss-programs

oss-programs is a simple app for finding open source opportunities. It shows mentorship programs, grants, fellowships, hackathons, and internships. The app helps you track deadlines, see stipend info, and export calendars for important dates. This tool is made for anyone interested in joining or learning about open source projects.

You don't need prior technical skills to use this app. It organizes all the information so you can easily explore and apply to programs that interest you.
